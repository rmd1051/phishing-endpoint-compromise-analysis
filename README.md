Phishing Attack Leading to Endpoint Compromise – SOC Investigation

This project documents a simulated Security Operations Center (SOC) investigation of a phishing attack that resulted in malware execution and potential system compromise.

📌 Objectives
Analyze phishing email activity
Identify Indicators of Compromise (IOCs)
Reconstruct attack timeline
Map findings to MITRE ATT&CK framework
Recommend mitigation strategies
🧠 Key Findings
Phishing email from a suspicious domain
Malicious file execution (flbpfuh.exe)
Registry modification indicating persistence
Outbound communication to external IP (91.185.23.222)
Possible privilege escalation via Administrator account
🕒 Timeline of Events
Email received
Malicious file downloaded
Registry modified
External connection established
User logoff and Administrator login
🛡️ Skills Demonstrated
Threat detection & analysis
Log analysis
Incident investigation
MITRE ATT&CK mapping
Security reporting
